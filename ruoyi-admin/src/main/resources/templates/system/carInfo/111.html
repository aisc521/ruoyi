中国海军节来临之际，055有新动作，或将和辽宁舰共同出席盛宴

空军世界

04-03 18:30

4月23日是我国的海军节，在这一盛大节日即将来临的时候中国海军有了新动作，国产万吨级驱逐舰055型驱逐舰第一次抵达青岛港，值得关注的是和辽宁舰停靠在了一起，也意味着我国海军的航母编队实力再次得以提升。对于中国海军来说055型驱逐舰的加入，大幅度的提高了饱和攻击的能力，而且在反舰和反舰等方面也有了很大的提升，这也为将来的整体作战能力打下了基础。

目前我国海军的航母编队主要的护航舰艇有052C和052D驱逐舰，此外还有054A护卫舰，随着055型驱逐舰服役部队之后，估计054护卫舰可能会逐渐从航母护航编队中撤出，也就是说我国海军的航母护卫舰将会全部升级为驱逐舰，因为这种舰艇的吨位大，载弹量更多作战能力也更加的强悍。

此外舰艇的垂直发射单元的数量也是一个重要的标准，据悉055型驱逐舰拥有112个垂直发射单元这已经超过了美军的伯克级驱逐舰，比提康德罗加巡洋舰稍微差一点，相比之下海军的052C型驱逐舰仅有48个垂直发射单元，还没有055发射单元的一半多，最重要的是052只能发射HQ9舰空导弹。

而且055型驱逐舰还拥有两个机库，能搭载两架舰载直升机而052驱逐舰仅仅只能搭载一架直升机，从未来的发展来看增加舰载直升机的数量是必然的取值，通过国内外的海军舰艇就能看得出来，对抗潜艇的能力和舰载反潜直升机的数量有着很大的关系。

最重要的是055型驱逐舰的很多能力是052驱逐舰所不具备的，最重要的就是反导能力，因为055长辈了大孔径的雷达，采用的是双波段综合射频技术，这大大的增强了055大驱的雷达探测能力，能够有效的拦截导弹。可以说装备055大驱之后的中国海军，在未来的海战中将会更加的所向披靡，同时也让我们更加的期待海军节上辽宁舰航母编队的共同亮相。
